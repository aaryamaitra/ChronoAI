# ⏱ ChronoAI - AI Powered Productivity & Time Management Portal

ChronoAI is a professional productivity web application designed to help users manage time, tasks, study goals, reminders, habits, and daily productivity using a clean dashboard and AI assistant.

## 🚀 Project Overview

ChronoAI combines time management tools with AI-based planning support. It helps students and professionals organize their daily work, track productivity, manage reminders, and generate study plans using an AI assistant.

## ✨ Features

- Professional landing page
- Login and signup system
- Dashboard with live clock
- Productivity analytics
- Task manager with priority and due date
- Notes module
- Habit tracker
- Calendar view
- Pomodoro timer
- Stopwatch
- Alarm and reminder system
- AI assistant using Gemini API
- Razorpay payment integration
- Subscription plans: Free, Pro, Premium
- Payment verification backend
- Profile page
- Settings modal
- Mobile responsive sidebar
- Active sidebar navigation

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### APIs / Services
- Gemini API for AI Assistant
- Razorpay API for payments

### Storage
- LocalStorage for frontend data
- Environment variables for API keys
